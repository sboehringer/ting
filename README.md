# Ting pen book downloading

Script to download books for a ting-pen [http://www.ting.eu]. Works as of 10/2019.

# Configuration

Change `tingf` to the folder location of the pen-mount. As configured, works with OpenSUSE (>= 15.0).

# Workflow

  * Insert, mount pen
  * Run `ting`
  * Unmount, use pen
